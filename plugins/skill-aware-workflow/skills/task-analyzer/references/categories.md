# タスクカテゴリ詳細定義

## Document（ドキュメント処理）

### 対象タスク
- Word文書の作成・編集
- PDF の作成・編集・フォーム入力
- Markdown ドキュメント作成
- テキスト抽出・変換

### キーワードパターン
```regex
(document|doc|docx|word|pdf|markdown|md|text|write|edit|report|letter|contract)
```

### 関連スキル
- docx: Word文書操作
- pdf: PDF操作
- markdown-tools: Markdown変換

---

## Presentation（プレゼンテーション）

### 対象タスク
- PowerPointスライド作成
- プレゼン資料デザイン
- スライドテンプレート適用

### キーワードパターン
```regex
(presentation|slide|pptx|powerpoint|deck|pitch)
```

### 関連スキル
- pptx: PowerPoint操作
- ppt-creator: スライド生成

---

## Data（データ処理）

### 対象タスク
- Excel/CSV データ処理
- データ分析・可視化
- データ変換・クレンジング

### キーワードパターン
```regex
(data|excel|xlsx|csv|spreadsheet|analysis|chart|graph|pandas|table)
```

### 関連スキル
- xlsx: Excel操作
- csv-tools: CSV処理

---

### キーワードパターン
```regex
(frontend|react|vue|angular|html|css|javascript|typescript|ui|ux|component|webpage|webapp|dashboard)
```

### 関連スキル
- frontend-design: UIデザイン
- react-components: Reactコンポーネント

---
